# INTDB INVESTOR DEMO - Interactive Questions

## How to Use This Demo

Open a new terminal, navigate to intdb folder, and start Claude Code:
```bash
cd ~/intdb
claude
```

Then just type these questions naturally. Claude will run live queries on your data.

---

## OPENING QUESTIONS (Start Here)

### Question 1: Overview
```
How many emails have you processed and what's the accuracy?
```

### Question 2: Live Activity
```
What's happening right now? Show me today's activity.
```

### Question 3: Live Feed
```
Show me the live email feed from the last few hours.
```
